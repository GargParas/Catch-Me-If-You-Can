# Catch-Me-If-You-Can
Catch Me If You Can is an interactive Simon Game written in JS/CSS/HTML to simulate the classic Simon Electronic Memory Game.

# Game Rule

Each level will have different set(s) of button blink, player must follow the order of how the button blink and click the corresponding button in order.
